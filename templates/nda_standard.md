## 使い方

1. Google Docs で新しいドキュメントを作成する
2. 以下の本文をコピーして貼り付ける
3. 必要に応じて書式や条項を調整する
4. gassign の「テンプレート管理」画面でこのドキュメントを登録する

秘密保持契約書

{{our_company_name}}（以下「甲」という）と{{counterparty_name}}（以下「乙」という）は、両者間の業務上の情報交換に関し、以下のとおり秘密保持契約（以下「本契約」という）を締結する。

第1条（目的）
本契約は、甲乙間で開示される秘密情報の取り扱いについて定め、情報の適正な保護を図ることを目的とする。

第2条（秘密情報の定義）
本契約において「秘密情報」とは、甲または乙が相手方に対して開示する一切の技術上、営業上、財務上その他の情報であって、書面、口頭、電磁的記録その他いかなる手段による開示かを問わず、秘密として指定されたものをいう。ただし、次の各号のいずれかに該当するものは秘密情報に含まれない。
（1）開示の時点で既に公知となっているもの
（2）開示後に受領者の責によらず公知となったもの
（3）受領者が独自に開発したもの
（4）正当な権限を有する第三者から秘密保持義務を伴わずに入手したもの
（5）法令または裁判所等の命令により開示を求められたもの

第3条（秘密保持義務）
甲および乙は、相手方から開示を受けた秘密情報を厳重に管理し、相手方の書面による事前の承諾を得ることなく、第三者に開示、漏洩または公開してはならない。

第4条（目的外利用の禁止）
甲および乙は、相手方から受領した秘密情報を、{{purpose}}（以下「本目的」という）以外の目的に使用してはならない。

第5条（複製の制限）
甲および乙は、相手方の書面による事前の承諾なく、秘密情報を複製してはならない。ただし、本目的の達成に必要な範囲内での複製はこの限りではない。

第6条（従業員等への開示）
甲および乙は、秘密情報を知る必要がある自己の役員・従業員（業務委託先を含む）にのみ開示することができる。この場合、開示を受けた者に対し、本契約と同等の秘密保持義務を負わせるものとする。

第7条（有効期間）
本契約の有効期間は、本契約締結日から{{contract_period}}間とする。ただし、有効期間満了の{{renewal_notice_period}}前までに甲乙いずれかから書面による更新拒絶の通知がない場合は、同一条件で自動更新されるものとし、以後も同様とする。

第8条（秘密情報の返還）
甲または乙は、相手方から書面による要求があったとき、または本契約が終了したときは、遅滞なく相手方の秘密情報（複製物を含む）を相手方に返還し、または廃棄するものとする。

第9条（損害賠償）
甲または乙が本契約に違反し、相手方に損害を与えた場合、違反した当事者は相手方に対し、その損害を賠償する義務を負う。

第10条（準拠法・管轄）
本契約は日本法に準拠し、本契約に関する一切の紛争については、{{jurisdiction}}地方裁判所を第一審の専属的合意管轄裁判所とする。

以上、本契約の成立を証するため、本書2通を作成し、甲乙各1通を保有する。

## テンプレート変数一覧

| 変数名 | 内容 | 例 |
|--------|------|----|
| `{{our_company_name}}` | 自社名（設定から自動） | sator株式会社 |
| `{{counterparty_name}}` | 相手先会社名 | 株式会社サンプル |
| `{{our_representative}}` | 自社代表者 | 代表取締役 瀬野陽介 |
| `{{counterparty_representative}}` | 相手先代表者 | 代表取締役 山田太郎 |
| `{{purpose}}` | 秘密情報の利用目的 | 新規事業に関する協議 |
| `{{contract_period}}` | 契約有効期間 | 1年 |
| `{{renewal_notice_period}}` | 更新拒絶通知期間 | 1か月 |
| `{{jurisdiction}}` | 管轄裁判所の所在地 | 東京 |
| `{{contract_date}}` | 契約締結日 | 2026年5月1日 |
